## 1. 假设陈述

> 给 v1 的 TSMOM 趋势信号加上**按品种波动率目标定仓**（高波动缩仓、低波动不加杠杆），
> 能把 v1 的灾难性回撤（MaxDD 68%）压回 <25%，并改善净 Sharpe。

**诚实预期（评估后写明，DEC-051）：** 波动率目标定仓主攻**回撤**（68% 的真 disqualifier）。
由于 Sharpe 对仓位常数缩放近似不变，v2 可能把 MaxDD 压达标、但**净 Sharpe 仍受资金费率拖累（COST-LIMITED 可能延续）**——资金费率/空头负贡献问题留给 v3（资金费率择向/过滤）。
故 v2 的关键学习点 = "回撤是否可由定仓解决"；联合门槛是否一次通过为次要。

## 2. 唯一变量与冻结设定（单变量）

**v1 全部冻结**：信号 sign(close[t]−close[t−540])、L=90天、多空双向、三品种 4H、t+1 开盘执行、
未翻转则持有、成本（手续费0.05%/边+滑点0.10%/边+真实8H资金费率按方向收付）、数据三分/Holdout截断、
1x 敞口口径（每根4H开盘检查并按比例校正，DEC-051）。

**唯一新变量 = 仓位定量：v1 等权满仓 → v2 按品种波动率目标定仓**
```
σ_i,t = 标的i 过去 M=180 根(30d) 对数收益的年化已实现波动率（仅用 ≤t 数据，无前视）
f_i,t = clip( τ / σ_i,t , 0 , cap_single )      # 该品种名义占组合比例
  τ = 0.20         # 每品种目标年化波动率（冻结，按波动率分布选定，非按盈亏）
  cap_single = 0.5 # 单品种名义上限
方向 = TSMOM 信号符号（不变）
gross 约束：Σ|f_i| ≤ 1.0（每根4H开盘检查，超出按比例缩减）；不加杠杆（不上调到>1x）
```
冻结常数：M=180、τ=0.20、cap_single=0.5、gross_cap=1.0、down-only（不加杠杆）。**禁止据结果调整。**

> 预热：σ 不足 180 根时该品种 f=0（不入场）。

## 3. 通过门槛（联合，冻结，与 v1 一致）

| 指标 | 门槛 | 口径 |
|------|------|------|
| Sharpe | > 1.0 | 三品种组合，扣全部成本含 funding |
| MaxDD | < 25% | 三品种组合 |

两项同时满足 → PASSED；任一不满足 → FAILED。

## 4. 机制诊断（非门槛，核心）

- **MaxDD：v2 vs v1（68.38%）是否显著下降、是否 <25%**（本实验首要学习点）。
- 净 Sharpe vs v1（0.720）；毛 Sharpe vs v1（1.043）。
- 平均/峰值 gross 敞口、缩仓触发频率（高波动期是否确实缩仓）。
- 品种权重分布（SOL 是否被显著压低；SOL 贡献变化）。
- 成本三项是否随敞口下降等比下降；资金费率是否仍为最大项（→ 是否仍 COST-LIMITED，指向 v3）。
- WF 三段 MaxDD/Sharpe；2022 是否仍正。

## 5. 失效判断（冻结）

- 联合门槛任一不满足 → FAILED。计数：TSMOM 家族实现细化，历史失败数 +1，**独立 Alpha 仍 5/8**（不增）。
- 若 MaxDD<25% 但净 Sharpe<1（资金费率所致）→ 记 "回撤已解决 / COST-LIMITED 延续"，直接motivate v3（资金费率择向）。
- 前视/单变量违规（改了 §2 任一冻结项）→ 作废重做。
