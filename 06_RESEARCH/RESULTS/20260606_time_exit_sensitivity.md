# 时间退出持仓期敏感性诊断

**日期：** 2026-06-06  
**性质：** 诊断分析，不是新假设闭环，不计失败  
**数据范围：** 固定前 80% `nrows`，Holdout 未读取  
**统一口径：** 同一实现运行 `t+6 / t+12 / t+24`；`t+12` 为严格同口径 v6 comparator

## 实现口径

- 冻结输入：`events_v5_sweep_regime_bull_v4.csv`
- 入场：信号后下一根 4H K线开盘，含 0.05% 滑点
- 时间退出：入场K线计为第 1 根，在第 N 根 4H K线收盘全仓退出
- 同根顺序：止损优先于时间退出
- 成本：双边 0.04% taker fee、双边滑点、真实 8H funding；缺失结算点按 0.01%
- 仓位：初始 100,000 USDT、每笔风险 1%、1x 总敞口约束
- 全部报告：三品种与 BTC+ETH，各自 `t+6 / t+12 / t+24`，共六组

## 结论

1. `t+24` 是六组中风险调整收益最强的持仓期，且相对 `t+12` 的日收益改善在探索性 bootstrap 中为正。
2. 优势并非免费：持有期越长，止损率从约 50% 上升到约 67%-68%，MaxDD 也升至 27%-38%。
3. `t+12` 严格 comparator 未通过成功条件；本任务仅诊断，不增加失败计数。
4. **建议，非决策：** `t+24` 可作为下一版预登记的候选退出期，但不应按当前仓位/止损结构直接升级；必须在运行前固定回撤控制变量，并重新做独立验证。
