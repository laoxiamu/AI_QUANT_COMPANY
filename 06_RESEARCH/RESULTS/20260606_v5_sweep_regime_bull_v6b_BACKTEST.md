## 止损距离与仓位机制

按三品种成交交易计算：

| 指标 | baseline | candidate |
|------|---------:|----------:|
| 平均 `entry-sweep_low` / 价格 | 2.33% | 2.33% |
| 平均 `entry-ref_low` / 价格 | 1.38% | 1.37% |
| 平均收紧距离 / 价格 | 0.95% | 0.96% |
| 收紧距离中位数 / 价格 | 0.59% | 0.60% |
| 平均初始名义仓位 | 99,365 USDT | 165,972 USDT |
| 被 1x 上限缩放的入场 | 134 | 246 |

`ref_low` 将风险距离平均从约 2.33% 缩短到 1.37%。在固定 1% 风险公式下，
这会先放大期望仓位，而不是缩小仓位；随后更多交易受到 1x 总敞口上限缩放。
因此收益变化来自“更紧止损 + 更高风险定仓需求 + 更频繁敞口缩放”的联合结果，
不能解释为更宽止损容纳了更多波动。

所有事件都满足 `sweep_low < ref_low`。两个版本 `entry <= stop` 的无效风险
拒绝数均为 0。
