# 结构确认Setup失败经验 v1

**来源实验：**

- `v5_sweep_choch_fvg_bull_v1`（4H）
- `v5_sweep_choch_fvg_bull_v2`（1H）

## 已验证事实

1. 4H实际交易158笔，样本不足且Sharpe、回撤、Expectancy不达标。
2. 1H实际交易621笔，解决样本不足，但验证集亏损9.15%，Walk-Forward由强正衰减至负。
3. 1H六组参数敏感性均未通过Sharpe、回撤和Expectancy门槛。
4. 同品种持仓冲突显著压缩候选：1H共同区间候选978个，最终只有621笔交易。
5. 组合累计收益为正不能替代验证集、Sharpe、回撤和稳定性门槛。

## 经验结论

- 当前失败不是单纯由4H样本少造成。
- 缩短周期增加样本，也增加噪声和时期依赖。
- 三重确认链混合Sweep、CHoCH、FVG、Regime和退出规则，难以识别哪个部分真正有信息。
- 后续应先做单事件条件收益研究，再逐个增加确认变量。
